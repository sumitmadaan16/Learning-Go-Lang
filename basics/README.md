# 📘 Basics

Covers the foundational building blocks of Go.

## Concepts Covered

| File | Concept |
|------|---------|
| `hello.go` | First Go program, formatted print |
| `variable_types.go` | int, float, bool, arrays, slices, maps, structs |
| `constants.go` | `const`, const expressions, `iota` enumerations |
| `scope.go` | Package, function, block, loop scope + closures |
| `pointers.go` | Pointer declaration, dereferencing, pointer-to-pointer |

## How to Run

```bash
cd basics
go run .
```
