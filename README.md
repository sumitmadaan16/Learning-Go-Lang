# 🧠 Learning-GoLang

A modular, hands-on repository to master Go (Golang) fundamentals — starting from the very basics and expanding into full backend and infrastructure readiness.

## 🚀 Current Progress

This repo currently includes:
- `Hello.go` – Your first Go program
- `VariableTypes.go` – Exploring Go's type system
- `Constants.go` – Working with immutable values

Each file focuses on a single concept to keep learning structured and incremental.

## 📚 Upcoming Topics

Planned additions include:
- Operators and expressions
- Control flow (if, switch, loops)
- Functions and parameters
- pointers
- Error handling and interfaces
- Concurrency and goroutines

> These will be added in future commits as the learning journey progresses.
